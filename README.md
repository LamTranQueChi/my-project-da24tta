- Form đăng nhập: Form đăng nhập với giao diện đơn giản.
- Form đăng kí: Form đăng kí tài khoản sử dụng HTML, CSS và JavaScript.
- Bảng hóa đơn: Hiển thị và xử lý bảng hóa đơn sản phẩm.
